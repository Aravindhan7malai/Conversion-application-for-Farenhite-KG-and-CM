Conversion Application 

This is a simple web application that performs unit conversions for temperature (Celsius to Fahrenheit), weight (Kilograms to Pounds), and length (Feet to Centimeters).

Link to check the page: https://aravindhan7malai.github.io/Conversion-application-for-Farenhite-KG-and-CM/

Usage

The application consists of three sections for different conversions:

1. **Celsius to Fahrenheit Conversion**:
- Enter the value in Celsius in the input field.
- Click the "Submit" button.
- The converted value in Fahrenheit will be displayed below.

2. **Kilograms to Pounds Conversion**:
- Enter the value in Kilograms in the input field.
- Click the "Submit" button.
- The converted value in Pounds will be displayed below.

3. **Feet to Centimeters Conversion**:
- Enter the value in Feet in the input field.
- Click the "Submit" button.
- The converted value in Centimeters will be displayed below.
